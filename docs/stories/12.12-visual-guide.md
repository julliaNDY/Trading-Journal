### 1. Sentiment Badge
- **Location**: Top left
- **Size**: Large (`text-lg px-6 py-2`)
- **Icon**: Dynamic based on sentiment
- **Color**: Matches sentiment (green/red/blue)

### 2. Confidence Display
- **Location**: Top right
- **Format**: "Confidence: XX%"
- **Style**: Medium font weight

### 3. Confidence Bar
- **Location**: Below badge/confidence
- **Height**: 2 units (`h-2`)
- **Color**: Matches sentiment
- **Animation**: Smooth transition

### 4. Citation Box
- **Location**: Below confidence bar
- **Background**: Muted (`bg-muted/30`)
- **Border**: Yes (`border-muted`)
- **Font**: Italic
- **Padding**: `px-4 py-3`

### 5. Main Text
- **Location**: Below citation
- **Width**: Constrained (`max-w-3xl`)
- **Alignment**: Centered on desktop
- **Line Height**: Relaxed (`leading-relaxed`)
