# Story 16.1: Mobile App Companion - Core Features

## Status

**Draft**

---

## Story

**As a** trader,  
**I want** a mobile app to track my trades on the go,  
**so that** I can stay connected to my trading journal.

---

## Acceptance Criteria

1. **AC1**: Application mobile (iOS + Android) avec features core.
2. **AC2**: Voir trades, dashboard, calendar.
3. **AC3**: Ajouter trades, notes, tags.
4. **AC4**: Sync avec app web (real-time).
5. **AC5**: Push notifications pour événements importants.

---
