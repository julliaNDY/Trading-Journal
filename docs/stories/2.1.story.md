## Status

**Draft**

## Story

**As a** platform engineer,  
**I want** to research and select market data providers for tick data,  
**so that** we can source historical tick data for Market Replay and backtesting.

---

## Acceptance Criteria

1. **AC1**: Research documente au moins 3 providers (Barchart, IBKR, Intrinio, etc.) avec coûts, qualité, coverage.
2. **AC2**: Product Manager notifié immédiatement pour chaque API identifiée (voir governance rules).
3. **AC3**: Budget validé pour au moins un provider prioritaire.
4. **AC4**: Provider sélectionné avec documentation API accessible.
5. **AC5**: Plan d'intégration (POC) défini pour le provider sélectionné.
