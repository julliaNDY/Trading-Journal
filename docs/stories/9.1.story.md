# Story 9.1: Landing Page - Hero & Value Proposition

## Status

**Draft**

---

## Story

**As a** visitor,  
**I want** a landing page with hero section and value proposition,  
**so that** I can understand what Trading Path Journal offers and sign up.

---

## Acceptance Criteria

1. **AC1**: Page `/` (landing) avec hero section et value proposition.
2. **AC2**: Hero section : visuels, headline, value proposition claire.
3. **AC3**: Features preview : carrousel des principales fonctionnalités.
4. **AC4**: Social proof : témoignages (si disponibles).
5. **AC5**: CTA : "Start Free Trial" / "View Pricing" (liens vers signup/pricing).
6. **AC6**: **"Join Discord" button** : Bouton Discord avec emphasis "Free" (lien externe vers Discord).
7. **AC7**: Performance : page load < 2s, SEO score > 90 (Lighthouse).
